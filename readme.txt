Sorting Algorithm Project

Description: 

This is a web-based tool that helps to visualize and compare the performace of different Sorting Algorithms. This 
can be used to compare the performace(execution time) of Sorting Algorithms such as Merge Sort, Quick Sort, Insertion Sort, Bubble Sort, 
Quick Sort using 3 Median, Selection Sort and Heap Sort for different input size which a user can dynamically type. Furthermore,
user can select a Algorithm from the list to check the execution time of a particular Sorting Algorithm for the input size user want.

Technologies Used:

1.) HTML
2.) CSS
3.) JAVASCRIPT
4.) CHART.JS Library for Graph Visualization

Features: 

1.) Choose a single algorithm to see the performance using a random number generator array
2.) Compare all Algorithms at once using a random number generator array 
3.) View a summarizing table for the runtime and input size for each Algorithm
4.) Using custom array input check the performace of the single algorithm
5.) Using custom array input check the performace of the algorithms at once
6.) Choose the user input size for array upto 100,000

How to run:

1.) Download all the files(index.html, algo.js, style.css) into a single folder
2.) Open index.html in the browser

Or

1.) Open the URL in browser: https://devarsh881.github.io/Algorithm-Project/ 

Usage: 

1.) Use the dropdown to select an algorithm 
2.) Input the array size for the random array generator or provide a custom array
3.) Run the algorithm using Run Single button to see the output for single algorithm using random array generator
4.) Use Compare All Algorithm button for output of all algorithms at once using random array generator
5.) Use Custom Array Input button for single algorithm using custom array input
6.) Use Compare All Using Custom Array button for output of all algorithms at once using custom array input
7.) Use Reset Graph button to clear the graph
